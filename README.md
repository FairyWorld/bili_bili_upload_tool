# bili_upload_tool
**搬运视频到b站工具**

## docker(推荐)
  [已上传至docker hub](https://hub.docker.com/r/sicko1111/sicko)
  **一键pull加run部署**
  ```
  docker pull sicko1111/sicko
  ```
  ```
  docker run -d --name mycontanier -p 9090:9090 sicko1111/sicko
  ```

Finally you can visit the website like <u>yoursite:9090</u>
